# IPTV App

This IPTV app is developed using Python and PyQt, providing a user-friendly interface to watch ongoing matches. The app leverages the Koora API to fetch and display live sports matches, ensuring a seamless viewing experience. With YallShoot, users can easily browse and watch their favorite matches with just a few clicks.

## Features

- Stream ongoing sports and more.
- User-friendly interface built with PyQt.
- Integration with Koora API for reliable and up-to-date channel listings.
- Easy navigation and channel selection.

## Requirements

- Python 3.x
- PyQt5
- Requests (for API integration)
